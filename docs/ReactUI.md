# Sensei React UI

## Install React

Install Node

```
$ brew update
$ brew install node
$ node -v
v11.5.0
```

Install React

```
$ npm install -g create-react-app
```

Build a React UI

```
$ cd ~/Rest
$ create-react-app react-ui
```

Start the react server

```
$ cd react-ui
$ npm start
```

Test the react server by browsing to [localhost:3000](http://localhost:3000).
